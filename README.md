I build small, practical systems around AI, crypto infrastructure, and web applications. My work tends to combine product design, backend automation, and frontend dashboards so ideas can be tested quickly and shipped cleanly.

- Working on AI-first apps, agents, and chat interfaces
- Building Solana tools for treasury automation and transparent on-chain flows

- Shipping with TypeScript, React, Next.js, Node.js, and API-driven backends
